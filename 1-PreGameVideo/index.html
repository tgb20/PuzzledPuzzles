INFO AND REQUIREMENTS

The game requires that you play on a Chrome Browser. You can play on another browser for a majority of the game, but at a later stage, if you're not using Chrome, a crucial feature of the game will not work. So you might as well switch over now.

The game is best played using a headset.

It's handy to have a pen and paper available

Some puzzles are better solved alone, some are better worked together.

The game is difficult. It takes most people 90-120mins to solve using 2-3 hints.
